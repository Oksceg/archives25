# Коллекция веб-архивов по киносценариям
Здесь представлена коллекция веб-архивов сайтов со сценариями фильмов и сериалов. Архивы были собраны в рамках курса по Цифровым Архивациям 2025

Инструменты, задействованные для работы:

- wpull для скачивания и веб-архивирования сайтов
- [archiveready](https://archiveready.com/) для анализа коллекции
- metawarc для описания метаданных
- [replayweb.page](https://replayweb.page/) для отображения собранных архивов

## Об архивированных сайтах

Сценарии фильмов и сериалов — прекрасный материал для исследований, связанных с поп-культурой. Наличие текстов в электронном виде и в свободном доступе позволяет осуществлять полноценные DH-исследования. Причинами для архивации подобных ресурсов можно считать уникальность коллекций и в то же время потенциальные блокировки, необновляемость сайтов, удаление данных. Так, в этой небольшой работе все 3 сайты обладают своими особенностями:

- [imsdb.com](https://imsdb.com) — крупнейшая бесплатная онлайн-база данных иностранных сценариев, обновляется до сих пор
- [moviescriptsandscreenplays.com](https://moviescriptsandscreenplays.com/) — небольшая библиотека сценариев, последнее обновление в 2007
- [kinoscenariy.com](https://kinoscenariy.com) — сценарии иностранных и российских фильмов, тексты есть и на русском, и на английском языках, обновляется до сих пор

Такую коллекцию можно пополнять, но стоит отметить: изначально в планах действительно подразумевалось собрать большее количество сайтов, но некоторые другие обкачивались слишком долго, что было связано, возможно, со структурой самих сайтов, а может и с объёмами данных. Поэтому дальнейшая работа должна происходить при наличии надёжной мощной техники и терпении от ожидания :)

## Как создавалась коллекция

Как было упомянуто в самом начале, для загрузки данных использовался инструмент wpull, представленный как библиотека Python. Для того, чтобы воспользоваться им, я установила его через терминал в дистрибутиве Anaconda.

Я уже достаточно давно знала сайт [imsdb.com](https://imsdb.com), и прежде всего, решила работать с ним. Далее мне было интересно, есть ли аналоги: в конце концов, даже в этой крупной базе данных представлен только ограниченный пласт произведений. В итоге найти и при этом более менее успешно обкачать удалось два других сайта: [moviescriptsandscreenplays.com](https://moviescriptsandscreenplays.com/) и [kinoscenariy.com](https://kinoscenariy.com).

## Структура и работа с данными
В данном репозитории есть 3 папки по 3 сайтам соответственно. В каждой находится следующая информация:

- Архивируемость сайтов, проверенная на специальном ресурсе [archiveready.com](https://archiveready.com/)

- Дополнительный анализ, произведенный с помощью утилиты [metawarc](https://github.com/datacoon/metawarc)

- Скриншоты главных старниц с сервиса [replayweb.page](https://replayweb.page/), которые частично отображают, насколько хорошо были заархивированы данные
