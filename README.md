# Коллекция веб-архивов по киносценариям
Здесь представлена коллекция веб-архивов сайтов с киносценариями, созданная в рамках курса по Цифровым Архивациям 2025

Инструменты, задействованные для работы:

- wpull для скачивания и веб-архивирования сайтов
- archiveready для анализа коллекции
- metawarc для описания метаданных

## Об архивированных сайтах

- imsdb.com — крупнейшая бесплатная онлайн-база данных иностранных сценариев, обновляется до сих пор
- moviescriptsandscreenplays.com — небольшая библиотека сценариев, последнее обновление в 2007
- kinoscenariy.com — сценарии иностранных и российских фильмов, тексты есть и на русском, и на английском языках, обновляется до сих пор
