# Коллекция веб-архивов по киносценариям
Здесь представлена коллекция веб-архивов сайтов с киносценариями, созданная в рамках курса по Цифровым Архивациям 2025

Инструменты, задействованные для работы:

- wpull для скачивания и веб-архивирования сайтов
- archiveready для анализа коллекции
- metawarc для описания метаданных

## Об архивированных сайтах

Сценарии фильмов и сериалов — прекрасный материал для исследований, связанных с поп-культурой. Наличие текстов в электронном виде и в свободном доступе позволяет осуществлять полноценные DH-исследования. Причинами для архивации подобных ресурсов можно считать уникальность коллекций и в то же время потенциальные блокировки, необновляемость сайтов, удаление данных. Так, в данной небольшой коллекции все 3 сайты обладают своими особенностями:

- [imsdb.com](https://imsdb.com) — крупнейшая бесплатная онлайн-база данных иностранных сценариев, обновляется до сих пор
- [moviescriptsandscreenplays.com](https://moviescriptsandscreenplays.com/) — небольшая библиотека сценариев, последнее обновление в 2007
- [kinoscenariy.com](https://kinoscenariy.com) — сценарии иностранных и российских фильмов, тексты есть и на русском, и на английском языках, обновляется до сих пор

Такую коллекцию можно пополнять, но стоит отметить: изначально в планах действительно было собрать большее количество сайтов, но некоторые другие обкачивались слишком долго, что было связано, возможно, со структурой самих сайтов, а может и с объёмами данных. Поэтому дальнейшая работа должна подразумевать наличие надёжной мощной техники и терпение при ожидании :)

## Как создавалась коллекция

Как было упомянуто в самом начале, для загрузки данных использовался инструмент wpull, представленный как библиотека Python. Для того, чтобы воспользоваться им, я установила его через терминал в дистрибутиве Anaconda.

Я уже достаточно давно знала сайт [imsdb.com](https://imsdb.com), и прежде всего, решила работать с ним. Далее мне было интересно, есть ли аналоги: в конце концов, даже в этой крупной базе данных представлен только ограниченный пласт произведений. В итоге найти и при этом более менее успешно обкачать удалось два других сайта: [moviescriptsandscreenplays.com](https://moviescriptsandscreenplays.com/) и [kinoscenariy.com](https://kinoscenariy.com).

## Дальнейшая работа
Архивируемость сайтов была проверена на специальном ресурсе [archiveready.com](https://archiveready.com/)

Дополнительный анализ был произведен с помощью утилиты [metawarc](https://github.com/datacoon/metawarc)

Отдельно результаты можно посмотреть в соответствующих папках по 3 сайтам. Всё вместе можно прочитать в этом гугл-документе
